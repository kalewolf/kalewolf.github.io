Here are a list of methods that might be useful for you to know.

Method Name Parameters What it Does
System.out.println() ("string") Prints a string
length() ("string") This tells you how many characters are in a string
charAt() (number to indicate what character of the word should be returned) Returns the character at a certain part of the string
split() ("character that the string should split at") Splits a string at the desired character
Integer.parseInt() ("string") Converts a string to an int
replace() ("character you want replaced", "character you are replacing with") Replaces character(s) in a string with a different character
floor() (int number) Rounds down number to the nearest integer
max() (integer, integer) Returns the integer of the highest value
min() (integer, integer) Returns integer of the lowest value
equals() ("string") Looks at 2 stings as returns true if they are equal
