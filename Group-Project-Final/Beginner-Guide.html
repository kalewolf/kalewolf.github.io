Arrays
Once you have a basic understanding of computer code the first thing that you will want to try is creating a variable. The way that you create a variable in Java is by first stating the variable type such as int, string, float, char, or boolean. Next you will state your variable name, and make sure that this name is understandable and concise so that it can be easily used later. Using this variable you can also create a print statement in order to return the value of your variable. In java a print statement follows the syntax System.out.println(); with your variable name in the paranthesis.
Because Java is an object oriented program, class structures can be intergral. A class structure is a piece of code used to create and describe attributes given to an object. For our example I will use dog breeds and the different traits that they have.
Arrays are a piece of code that can be used to store a list of data. This is commonly utilized in lists of stored values or data sets that need to be grouped. Creating a Array follows the syntax of value_type array_name[] = new value_type[# of items in the array] and changing the value of an item in an array follows the syntax array_name[# corresponding to item in the list] = new_value.
