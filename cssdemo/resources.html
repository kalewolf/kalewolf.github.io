PNG Images

PNG Images Wikipedia

.png is a file format that stores data for an image in either RGB or RGBA (A stands for alpha channel that controls transparency).

  • This file format is for easy image transfering over the internet while retaining a certain amount of quality
  • It is the most common form of picture file format.
  • This file format cannot use professional grade image and color quality.
  • This file format cannot support CMYK (Cyan, Magenta, Yellow, Key).

JPEG Images

JPEG Images Wikipedia

.jpeg is a file format that is used most commonly as a compression method by trading image quality for storage space.

  • This file format is great at saving storage space that an image takes up.
  • This file format reduces image quality.
